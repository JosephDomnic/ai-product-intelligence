# AI Product & Competitive Intelligence

AI-powered platform that analyses customer reviews and competitors 
to generate data-driven product strategy.

## Features
- Customer review analysis with prioritised product backlog
- Competitor intelligence with feature comparison table
- AI-generated 90-day strategic roadmap
- Downloadable strategy report

## How to run
1. `pip install -r requirements.txt`
2. Get free API key from https://console.groq.com
3. Copy `.env.example` to `.env` and add your key
4. `streamlit run app.py`

## Tech Stack
Python • Groq API (Llama 3.3 70b) • Streamlit • Trafilatura • BeautifulSoup
